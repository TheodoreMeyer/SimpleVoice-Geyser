# Simple Voice Geyser Docs

This folder contains release-focused documentation for Simple Voice Geyser changes that are not fully covered by the public website yet.

## PR #45 / 0.1.2 Topics

- [Audio transport](audio-transport.md): `legacy` vs `svg-v2`, capability negotiation, fallback behavior, and the binary frame layout.
- [Configuration and upgrades](configuration-and-upgrades.md): config migration, backups, `/svg reload`, and currently inactive config values.
- [Proxy and subpath deployment](proxy-and-subpath.md): `server.context-path`, root/subpath hosting, and websocket proxy requirements.

These pages supplement the existing website pages for installation, security, proxy setup, commands, and API notes. They are intended to help server owners understand the behavior introduced around the 0.1.2 release cycle.
